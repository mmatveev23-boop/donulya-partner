## Единая база — amoCRM

Google Sheet убран. amoCRM = единая база для партнёров и лидов.

### Воронка «Партнёры»

| Этап | Описание |
|------|----------|
| 🔵 Новый партнёр | Прошёл авторизацию через Salebot |
| 🟡 На обучении | Проходит v2.html |
| 🟢 Активный | Получил ref-ссылку, передаёт лиды |
| 🟣 Топ-партнёр | 5+ договоров |
| ⚫ Неактивный | Нет лидов 30+ дней |

### Поля контакта «Партнёр»

| Поле | Тип | Пример |
|------|-----|--------|
| Имя | Текст | Иван Петров |
| Телефон | Телефон | +79121234567 |
| Мессенджер | Список | TG / VK / MAX |
| ID в мессенджере | Текст | chat_id для уведомлений |
| **Ref-код** | **Текст** | **IVAN482** |
| **Ref-ссылка** | **Текст** | **donula.online/ref?ref=IVAN482** |
| Уровень | Список | Новичок / Профи / Топ / Легенда |
| Всего лидов | Число | Автосчёт |
| Всего договоров | Число | Автосчёт |
| Баланс к выплате | Число | Автосчёт |

### Воронка «Партнёрские лиды»

| Этап | Цвет | Описание |
|------|------|----------|
| Новый лид | 🔵 | Заявка с ref.html или от партнёра напрямую |
| Квалификация | 🟡 | Юрист проверяет: долг ≥250к, подходит ли |
| Консультация назначена | 🟠 | Звонок/встреча запланирована |
| Консультация проведена | 🟣 | Юрист провёл, клиент думает |
| Договор заключён | 🟢 | Клиент подписал → партнёру +10 000 ₽ |
| Выплата партнёру | ✅ | Вознаграждение выплачено |
| Отказ | 🔴 | Клиент отказался / не подходит |

### Поля сделки «Партнёрский лид»

| Поле | Тип | Описание |
|------|-----|----------|
| Имя клиента | Текст | Из формы ref.html |
| Телефон клиента | Телефон | Из формы |
| Сумма долга | Список | 250-500к / 500к-1М / 1-3М / 3М+ |
| Мессенджер клиента | Список | TG / VK / MAX / Телефон |
| **Ref-код партнёра** | **Текст** | **IVAN482** |
| Канал привлечения | Список | Ссылка / Прямой контакт |
| Статус выплаты | Список | Не выплачено / Выплачено |
| Сумма выплаты | Число | 10000 |

### Автоматизации в amoCRM

1. **Новый лид** → автозадача юристу «Позвонить за 30 мин»
2. **Договор заключён** → уведомление партнёру в мессенджер + обновить баланс
3. **Нет активности 2 дня** → напоминание юристу
4. **Отказ** → уведомление партнёру «Клиент не подошёл»
5. **Новый партнёр** → генерация ref-кода → заполнение поля ref-ссылки

### Каналы связи в amoCRM

| Канал | Интеграция | Для чего |
|-------|-----------|----------|
| Телефония | Мегафон / МТТ / Sipuni / OKI-TOKI | Звонки юристов, запись |
| Telegram | Salebot / Wazzup | Переписка с клиентами и партнёрами |
| ВКонтакте | Salebot / amoCRM встроенный | Сообщения |
| MAX | Salebot | Авторизация и уведомления |

## Интеграция ref.html → amoCRM

```
ref.html форма (шаг 1 + шаг 2)
    → POST webhook на amoCRM API (или через Salebot/n8n)
    → Создаёт сделку в воронке «Партнёрские лиды»
    → Этап «Новый лид»
    → Заполняет поля: имя, телефон, долг, мессенджер, ref-код
    → Находит партнёра по ref-коду → привязывает к сделке
    → Уведомление партнёру: "🔔 Новый лид от вашей ссылки!"
```

## Salebot — авторизация партнёра (сделано Иваном)

### Файлы на сервере (donula.online/partners/)
- `api/session.php` — API сессий (создание, поллинг, подтверждение)
- `api/webhook.php` — обработчик Salebot (TG/VK/MAX, запрос телефона)
- `index.html` — фронтенд авторизации с ?step=N

### Процесс авторизации
```
v2.html экран 4 → кнопка TG/VK/MAX
    → donula.online/partners/index.html (авторизация)
    → Salebot создаёт сессию
    → В мессенджере: "Отправьте номер телефона"
    → Партнёр отправляет номер
    → Salebot подтверждает сессию
    → Создаёт контакт в amoCRM (воронка «Партнёры»)
    → Генерирует ref-код (первые 4 буквы имени + 3 цифры)
    → Заполняет поля: ref-код, ref-ссылка, мессенджер
    → Возвращает на v2.html?step=5 (обучение продолжается)
```

## Генерация ref-кодов

Автоматически при создании контакта в amoCRM:
- Первые 4 буквы имени в транслите + 3 случайные цифры
- Иван → IVAN482, Сергей → SERG117, Ольга → OLGA935
- Проверка уникальности по базе amoCRM
- Ref-ссылка: `donula.online/ref?ref=IVAN482`

## Бот-ЛК партнёра (через Salebot)

Salebot = и авторизация, и ЛК. Партнёр общается с ботом в мессенджере.

### Команды бота

| Команда | Что делает |
|---------|-----------|
| /start | Регистрация нового партнёра |
| Моя ссылка | Показать реферальную ссылку |
| Мои лиды | Список лидов с статусами (из amoCRM) |
| Мой баланс | Сколько заработано / к выплате |
| Передать номер | Отправить контакт клиента |
| Готовые посты | Шаблоны для рассылки |
| Рейтинг | Лидерборд партнёров |

### Передача номера через бота
```
Партнёр: "Передать номер"
Бот: "Введите имя и телефон клиента"
Партнёр: "Сергей +79031234567"
Бот: "✅ Принято! Лид 'Сергей' добавлен.
      Юрист свяжется с ним в течение 30 минут."
    → Создаёт сделку в amoCRM (канал: прямой контакт)
    → Привязывает к партнёру
```
